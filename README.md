WFM Analytics Reporting Automation

Proyecto desarrollado para automatizar la extracción y análisis de métricas operativas en entornos de contact center utilizando la API de Genesys Cloud.

🚀 Objetivo
Automatizar el proceso de reporting operativo, reduciendo tiempos manuales y mejorando la visibilidad de métricas clave.

🧠 Tecnologías
- Python
- Genesys Cloud API
- Data Processing

 📊 Funcionalidades
- Extracción de datos desde API
- Procesamiento de métricas (NDA, abandonos, volumen)
- Cálculo de disponibilidad (Avail)
- Generación de reportes estructurados

💡 Impacto
- Mejora en tiempos de análisis
- Automatización de tareas manuales
- Mayor precisión en datos operativos

🔧 Próximos pasos
- Integración con dashboards (Power BI)
- Automatización diaria
